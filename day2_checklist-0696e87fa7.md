## 📝 今日学习笔记

### 核心概念记录
- **列表推导式优势**：代码简洁、执行效率高
- **字典查找效率**：O(1)时间复杂度，适合快速查找
- **异常处理原则**：尽可能精确捕获异常，避免使用裸except
- **二分查找**：有序数组查找的高效算法，时间复杂度O(log n)

### 代码模板记录
```python
# 列表推导式模板
result = [表达式 for 变量 in 可迭代对象 if 条件]

# 字典推导式模板
result = {key: value for key, value in iterable}

# 异常处理模板
try:
    # 可能出错的代码
except ExceptionType as e:
    # 异常处理
finally:
    # 无论是否出错都会执行
```

### LeetCode解题思路
1. **加一**：从右向左处理进位，最高位进位需在数组前加1
2. **搜索插入位置**：二分查找，找不到时left就是插入位置
3. **合并有序数组**：从后向前合并，避免覆盖未处理的元素
